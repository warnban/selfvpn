## Зачем второй VPS? Можно ли в РФ?

### Зачем отдельная машина для бота и панели

| Причина | Объяснение |
|---------|------------|
| **Безопасность** | Бот смотрит в интернет, принимает фото, хранит БД. Если его взломают — VPN-сервер отдельно, ключи изолированы |
| **Стабильность** | Бот и cron не нагружают VPN-сервер |
| **Удобство** | Панель и бот можно перезапускать, не трогая VPN |

### Обязателен ли второй VPS?

**Нет.** На старте можно:

1. **Дешёвый VPS в РФ** (~200–400 ₽/мес) — **отлично подходит** для бота и панели
2. **Домашний ПК / Windows** — для тестов (бот работает, пока ПК включён)
3. **На том же NL VPS** — можно, но хуже с точки зрения безопасности

### NL или РФ для второго сервера?

| | NL VPS (VPN) | VPS для бота |
|--|--------------|--------------|
| **Нужна страна** | Да, NL (или другая «чистая» для VPN) | **Не важно** |
| **Telegram API** | — | Работает из РФ без проблем |
| **SSH к NL VPN** | — | Работает из РФ без проблем |

**Вывод:** второй VPS **можно брать в РФ** (Timeweb, Selectel, Aeza RU и т.д.). NL нужен только для VPN-трафика.

### Минимальный бюджет

- NL VPS (VPN): уже есть
- RU VPS (бот + панель): от ~200 ₽/мес **или** бесплатно на домашнем ПК для теста

# ЭТАП 1 — Telegram-бот (15 минут)

## Шаг 1.1 — Создай бота в Telegram

1. Открой Telegram, найди **@BotFather**
2. Отправь `/newbot`
3. Имя бота (отображаемое): например `My VPN NL`
4. Username (латиница, заканчивается на `bot`): например `myvpn_nl_bot`
5. **Сохрани токен** вида `7123456789:AAH...` — это `BOT_TOKEN`

## Шаг 1.2 — Узнай свой Telegram ID

1. Найди **@userinfobot** или **@getmyid_bot**
2. Нажми Start
3. **Сохрани число** (например `123456789`) — это твой `ADMIN_IDS`

## Шаг 1.3 — Настрой проект на Windows

```powershell
cd C:\selfvpn
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
copy .env.example .env
```

Открой `.env` в блокноте и заполни минимум:

```env
BOT_TOKEN=твой_токен_от_BotFather
ADMIN_IDS=твой_telegram_id
DAILY_PRICE_RUB=5
TRIAL_DAYS=2
REFERRAL_BONUS_RUB=5
PAYMENT_CARD=2200 ... твоя карта
PAYMENT_BANK=Сбер
PAYMENT_HOLDER=Имя Ф.
```

Панель пока можно не трогать — бот запустится без VPN.

## Шаг 1.4 — Первый запуск бота

```powershell
python -m bot.main
```

Должно появиться: `Бот запущен`.

1. Открой своего бота в Telegram → `/start`
2. Проверь меню, баланс (должно быть 10 ₽ при тарифе 5 ₽/день и 2 дня триала)
3. Рефералы — скопируй ссылку, открой в другом аккаунте/у друга — рефереру +5 ₽

# ЭТАП 2 — Amnezia Web Panel (30–45 минут)

Панель управляет твоим **существующим** `amnezia-awg2` по SSH.

## Шаг 2.1 — Где ставить панель

**Вариант A (рекомендую для продакшена):** дешёвый VPS в РФ  
**Вариант B (для теста сейчас):** на Windows рядом с ботом

### Вариант B — панель на Windows

```powershell
git clone https://github.com/PRVTPRO/Amnezia-Web-Panel.git C:\amnezia-panel
cd C:\amnezia-panel
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

Открой http://localhost:5000 → логин `admin` / `admin` → **сразу смени пароль**.

## Шаг 2.2 — Добавь NL VPN-сервер в панель

1. **Servers → Add Server**
2. IP твоего NL сервера
3. Login: `root`
4. Port: `22`
5. Password или SSH-ключ
6. Дождись проверки — должен появиться **AmneziaWG 2.0** / `awg2`

## Шаг 2.3 — Создай API-токен

1. **Settings → API Tokens → Create**
2. Скопируй токен `awp_...` (показывается один раз!)
3. В `.env` бота:

```env
PANEL_URL=http://127.0.0.1:5000
PANEL_API_TOKEN=awp_...
PANEL_SERVER_ID=0
PANEL_PROTOCOL=awg2
```

## Шаг 2.4 — Проверка API вручную

```powershell
curl -X POST "http://127.0.0.1:5000/api/servers/0/connections/add" `
  -H "Authorization: Bearer awp_ТВОЙ_ТОКЕН" `
  -H "Content-Type: application/json" `
  -d "{\"protocol\":\"awg2\",\"name\":\"test_user\"}"
```

В ответе должны быть `vpn_link` и `client_id`.

## Шаг 2.5 — Перезапусти бота и проверь VPN

```powershell
python -m bot.main
```

В боте: **🔐 Подключить VPN** → должна прийти ссылка `vpn://...`

Вставь в Amnezia на телефоне → проверь подключение.

# ЭТАП 4 — Бизнес-логика (уже в коде)

| Функция | Как работает |
|---------|--------------|
| **Пробный период** | При первом `/start` баланс = `DAILY_PRICE_RUB × TRIAL_DAYS` |
| **Тариф** | Каждый день в 03:00 MSK списывается `DAILY_PRICE_RUB` |
| **Кончился баланс** | VPN-клиент удаляется на сервере |
| **Рефералка** | Ссылка `t.me/bot?start=ref_TELEGRAM_ID` → новый юзер → рефереру `REFERRAL_BONUS_RUB` |
| **Оплата** | Пользователь шлёт сумму + скрин → тебе кнопки ✅/❌ |

# ЭТАП 5 — Что настроить под себя

В `.env`:

```env
DAILY_PRICE_RUB=5      # 5 ₽/сутки = 150 ₽/мес
TRIAL_DAYS=2           # 2 дня бесплатно = 10 ₽ на балансе
REFERRAL_BONUS_RUB=5   # 1 день в подарок рефереру
```

Примеры:
- 10 ₽/день, 2 дня триала → стартовый баланс 20 ₽
- Реферал 10 ₽ → друг зашёл = тебе 2 дня бесплатно
