Акции и пополнения

Множитель пополнения: при включённой акции пользователь получает на баланс больше, чем заплатил. Например, при ×2 пополнение на 100 ₽ даёт 200 ₽ на баланс.

Сейчас: {% if deposit_multiplier > 1 %} акция ×{{ deposit_multiplier|int }} активна — пополнение на 100 ₽ = {{ (100 * deposit_multiplier)|int }} ₽ на балансе. {% else %} акция выключена — начисляем ровно сумму пополнения. {% endif %} Применяется к оплатам картой/СБП и Telegram Stars. Ручные начисления админом не удваиваются.

Пользователь не сможет пополнить меньше этой суммы. При тарифе {{ daily_price|int }} ₽/сутки это ≈ {{ (min_topup / daily_price)|round(0, 'ceil')|int if daily_price else 0 }} дн. минимум.

Telegram Stars

Тариф в рублях задаётся в .env (DAILY_PRICE_RUB). Здесь — сколько Stars платит пользователь за 1 день сервиса в боте.

Пример: 30 дн. = {{ stars_per_day * 30 }} ⭐ (≈ {{ daily_price * 30 }} ₽ на балансе)
